# LMPD Crime Analytics
This project will analyze data extrapolated from the LMPD Crime database. It shows a few visualizations designed to help display the data in a way that is communicative and reliable.

# Instructions
1. Clone this repository
2. Create a virtual environment (venv)
3. Install the following packages
   - Matplotlib.pyplot
   - Pandas
   - Numpy
   - Seaborn
